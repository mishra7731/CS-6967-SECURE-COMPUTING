# 🔐 CS 6967 – Security Operations Lab Portfolio

This repository contains hands-on cybersecurity labs and threat hunting exercises completed as part of **CS 6967: Security Operations** at the University of Utah.

The work demonstrates practical experience in penetration testing, network security, intrusion detection, and digital forensics using real-world tools and simulated environments.

---

## 🎯 Overview

This project simulates an end-to-end security lifecycle, including:

- Reconnaissance and network scanning  
- Exploitation of vulnerabilities  
- Intrusion detection and monitoring  
- Threat hunting across network, memory, and disk  
- Forensic analysis of compromised systems  

---

## 🧪 Lab Coverage

###  Scanning

- Identified active hosts in LAN using `nmap`  
- Discovered open ports and services  
- Understood subnetting and network structure  

---

###  Exploitation & Attacks

- Brute force attack using Burp Suite  
- SQL injection using SQLMap  
- HTTP request interception and manipulation  

---

###  Network Security & Firewalls

- Configured WAN ↔ LAN access rules  
- Implemented firewall policies  
- Tested port forwarding and access restrictions  

---

###  Intrusion Detection

- Monitored malicious traffic using `tcpdump`  
- Detected SQL injection attacks using Snort rules  
- Analyzed attack payload patterns  

---

###  Secure Network Infrastructure

- Configured VPN server within LAN  
- Established SSH-based secure communication  
- Implemented jump server for controlled access  

---

###  System & Data Protection

- Created backup server with `rsync`  
- Simulated ransomware attack and recovery  
- Verified integrity of transferred data  

---

###  Threat Hunting & Forensics

####  Memory Forensics
- Captured memory dump using WinPMEM  
- Analyzed processes and system info with Volatility3  
- Investigated active network connections  

####  Network Forensics
- Analyzed PCAP files using Wireshark & Tshark  
- Identified suspicious traffic patterns  
- Investigated DHCP and NetBIOS activity  

####  Disk Forensics
- Created disk images using FTK Imager  
- Analyzed filesystem using Autopsy  
- Verified integrity via hash validation  

---

##  Tools & Technologies

- **Network & Scanning:** Nmap, Wireshark, Tshark  
- **Exploitation:** SQLMap, Burp Suite  
- **Monitoring:** Tcpdump, Snort  
- **Forensics:** Volatility3, Autopsy, FTK Imager  
- **Systems:** Kali Linux, Windows VM, OpenWRT  
- **Networking:** VPN, SSH, NAT, Port Forwarding
